Program 1:String into Primitive Data Types

package PracticalProjects;

public class TypeCasting {
	public static void main(String[] args) {
		System.out.println("Implicit Type Casting");
		char ch='a';
		System.out.println("The Character is "+ ch);
		int i=ch;
		System.out.println("The converted integer form is "+i);
		long l=ch;
		System.out.println("The long form is "+ l);
		double d=ch;
		System.out.println("The Double form is "+ d);
		
		
		System.out.println("Explicit Type Casting");
		double s=69;
		System.out.println("The double is "+s);
		int in=(int)s;
		System.out.println("The integer is "+in);
		char c=(char)s;
		System.out.println("The character is "+c);
		

	}


}

Output:

Implicit Type Casting
The Character is a
The converted integer form is 97
The long form is 97
The Double form is 97.0
Explicit Type Casting
The double is 69.0
The integer is 69
The character is E
