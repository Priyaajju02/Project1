Program 9:Conversion of String to String Buffer and String Builder

package PracticalProjects;

public class ConversionOfString {
	public static void main(String[] args) {
		System.out.println("Conversion of String to String Buffer and String Builder");
		System.out.println("\n");
		String str="Hello";
		StringBuffer a=new StringBuffer(str);
		a.reverse();
		System.out.println("String to StringBuffer:");
		System.out.println(a);
		StringBuilder b=new StringBuilder(str);
		b.append("World");
		System.out.println("String to StringBuilder:");
		System.out.println(b);
		
	}

}

Output:

Conversion of String to String Buffer and String Builder


String to StringBuffer:
olleH
String to StringBuilder:
HelloWorld
