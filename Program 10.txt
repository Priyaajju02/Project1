Program 10:Single and Multi Dimensional Arrays

package PracticalProjects;

public class Arrays {
	public static void main(String[] args) {
		//Single dimensional array
		System.out.println("Single Dimensional Array");
		System.out.println("\n");
		int a[]= {1,2,3,4,5};
		for(int i=0;i<5;i++) {
			System.out.println("Elements of Array a:"+a[i]);
		}
		//Multi dimensional array
		System.out.println("Multi Dimensional Array");
		System.out.println("\n");
		int [][]b= {{6,7,8,9},{10},{11,12,13}};
		System.out.println("Length of row 1: " + b[0].length);
        System.out.println("Length of row 2: " + b[1].length);
        System.out.println("Length of row 3: " + b[2].length);
	}

}

Output:

Single Dimensional Array


Elements of Array a:1
Elements of Array a:2
Elements of Array a:3
Elements of Array a:4
Elements of Array a:5
Multi Dimensional Array


Length of row 1: 4
Length of row 2: 1
Length of row 3: 3
